02. HTML 基础

HTML（HyperText Markup Language，超文本标记语言）是构建网页的标准标记语言。

一、什么是 HTML

HTML 是用于创建网页结构的标准 markup 语言，它使用标签来描述网页的内容和结构。

特点

  • 标记语言：使用标签来组织内容
  • 层次结构：标签可以嵌套
  • 语义化：标签有明确的含义
  • 跨平台：所有浏览器都支持

常用属性

属性 说明 示例
id 唯一标识 <div id="content">

style 内联样式 <div style="color: red">
title 提示文本 <div title="提示">
alt 替代文本 <img alt="描述">
href 链接地址 <a href="url">
src 源地址 <img src="url">
