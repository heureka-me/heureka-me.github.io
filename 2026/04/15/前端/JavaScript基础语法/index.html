1.1 JavaScript 基础语法

JavaScript 是一种动态、弱类型的编程语言，广泛应用于网页开发、Node.js 后端等领域。

一、JavaScript 简介

特点

  • 动态类型（无需声明类型）
  • 脚本语言（解释执行）
  • 面向对象（基于原型）
  • 函数式编程支持
