1.4 DOM 操作

DOM（Document Object Model，文档对象模型）是 HTML 文档的编程接口，JavaScript 通过 DOM 与 HTML 页面进行交互。

一、DOM 概述

什么是 DOM

DOM 是 HTML 文档的树状结构表示，JavaScript 可以通过 DOM 访问和修改页面内容。

DOM 节点类型

  • 文档节点：document（整个文档的根节点）
  • 元素节点：HTML 标签（<div>、<p> 等）
  • 文本节点：元素内的文本内容
  • 属性节点：元素的属性（id、class 等）
  • 注释节点：HTML 注释
